After Sol LeWitt

This example of a generative pattern is inspired by Sol LeWitt's Wall Drawings. To change the paramaters that affect the outcome of the generative code, see the comments in the sketch.js file. You can alter the size in pixels of the grid, the number of columns and rows, the colours as well as the probability of colours appearing. After you have edited the sketch.js file, save it, and then refresh this browser window to see the changes. You can then click your mouse onto the pattern to see different outcomes the code can produce.
