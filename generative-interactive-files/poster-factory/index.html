Poster Factory

This is a coded version of the poster factory algorithm that you and your fellow students carried out in class. To change the paramaters that affect the outcome of the generative code, see the comments in the sketch.js file. You can alter the the number of times it 'loops' through the set of steps to create more simple or more complex outcomes. After you have edited the sketch.js file, save it, and then refresh this browser window to see the changes. You can then click your mouse onto the pattern to see different outcomes the code can produce.
